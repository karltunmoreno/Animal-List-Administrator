A Zoo Animal Catalog

Portal to the Animal World.

Dive inside this wild animal application that uses the CRUD Restful Api Methods to validate query routes in Insomnia. Now that everything work by using express routes through the use of the Node server brought to you by the Javascript server machine, we can organize and view our animal catalog. Here we can categorize animals to their species class. Assign the foods that these animals eat in their daily diet. Also, we can add new animals to the list with their unique personality type. As well, we added space for the Zoo Keeper with age, and favorie animal preference. A warm, happy, fuzzy application.
